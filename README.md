## About LandCraft

LandCraft is a modular landscape material system for Unreal Engine 5, designed to give artists full control over terrain surface complexity without sacrificing performance.

LandCraft is part of the **MasterCraft framework** — the complete layered material ecosystem containing LayerCraft & LandCraft.

Built around UE5's native Material Layer framework, LandCraft requires no plugins, no external dependencies, and no engine modifications. It is a pure asset package — drop it into any project and it works immediately, regardless of project type or scale.

Stack, blend, and customize landscape material layers non-destructively — from simple terrain surfaces to complex multi-layer environments with procedural masking, slope and elevation blending, paint-driven layers, runtime virtual texture support, and grass map integration. Lightweight by design, LandCraft integrates cleanly into existing pipelines without disrupting your current setup.

Whether you are building open-world environments, architectural visualizations, or cinematic scenes, LandCraft provides a consistent, scalable workflow that grows with your project.
