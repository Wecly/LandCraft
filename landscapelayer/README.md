# ML\_LandscapeLayer

**Layer Asset Reference**

![](<../images/image8.jpg>)

`ML_LandscapeLayer` defines how a single terrain surface looks and behaves. Every visual property of a landscape layer — textures, UV logic, color, physical response, detail normals, deformation, surface variation, and grass density — is controlled through the parameter groups below.

```
Layer  →  UV Logic  →  Textures  →  Color  →  Response  →  Detail  →  Deform  →  Variation  →  Grass Maps
```
