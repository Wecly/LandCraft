# 🔍 Detail

![](<../images/image30.jpg>)

Adds a secondary detail normal layer to increase surface micro-detail at close range without additional albedo cost.

---

| Parameter | Description | Default |
|-----------|-------------|---------|
| **Normal Intensity** | Blending intensity of the base normal map. | `1.0` |

---

![](<../images/image12.jpg>)

### ♦ use Detail Normal

Enables a secondary detail normal map overlaid on the base normal for fine micro-surface detail.

| Parameter | Description | Default |
|-----------|-------------|---------|
| **use Detail Normal Rotation** | Enables UV rotation for the detail normal. | — |
| **Detail Normal Intensity** | Blend strength of the detail normal. | `0.0` |
| **Detail Normal Tiling** | UV tiling scale of the detail normal. | `1.0` |
| **Detail Normal Rotation** | Rotation angle applied to detail normal UVs. | `0.0` |
| **Detail Normal Texture** | Detail normal map texture asset. | `T_FlatNormal` |
