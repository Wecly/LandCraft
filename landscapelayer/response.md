# 💎 Response

![](<../images/image26.jpg>)

Controls the surface's physical light response — specular intensity and roughness adjustments determining how the surface reflects light.

---

| Parameter | Description | Default |
|-----------|-------------|---------|
| **Specular** | Specular intensity. `0.5` is the PBR default for non-metallic surfaces. | `0.01` |
| **Roughness Amount** | Additive roughness offset applied over the texture roughness. | `0.0` |
| **Roughness Contrast** | Contrast multiplier on the roughness map. | `1.0` |
