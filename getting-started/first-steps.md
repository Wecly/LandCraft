# First Steps

1. Create a new **Landscape Material** or open an existing one.
2. Set up landscape material layers using `ML_LandscapeLayer` instances as Layer assets.
3. Assign an `MLB_LandscapeBlender` as the **Blend Asset** for each layer transition.
4. Adjust per-layer parameters using the grouped parameter system described in this reference.
5. Use the **Debug** parameter in any layer to visualize individual material channels during setup.
